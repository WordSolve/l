# Examples

## Basic Usage

### List all words
```bash
$ python3 l.py list
```

### Search for specific words
```bash
$ python3 l.py search "light"
# Output: light, delightful

$ python3 l.py search "love"
# Output: love, lovely
```

### Get a random word (great for inspiration!)
```bash
$ python3 l.py random
# Output: luminous (or any random word from the collection)
```

### Count total words
```bash
$ python3 l.py count
# Output: Total words: 46
```

## Creative Use Cases

1. **Daily Inspiration**: Run `python3 l.py random` each morning for a word of the day
2. **Word Games**: Use the search feature to find words for puzzles
3. **Writing Prompts**: Get random words for creative writing exercises
4. **Learning Tool**: Explore interesting words containing the letter 'l'

## Philosophy

The `l` utility embraces minimalism:
- One letter focus
- Simple commands
- Clear output
- Zero complexity

Do what you want with it!
