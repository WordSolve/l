Complete implementation of a multi-dimensional computational cryptocurrency mining dashboard featuring quantum-enhanced mining algorithms for Bitcoin, Monero (XMR), and RedCode Coin (RDC).

## How to Use

### Quick Start
```bash
# Install dependencies
pip install -r requirements.txt

# Run dashboard
python miner_dashboard_ui.py

# Or use startup script
./start_dashboard.sh          # Linux/Mac
start_dashboard.bat           # Windows
```

### Test the System
```bash
# Core engine test
python miner_core.py

# Comprehensive test suite
python test_dashboard.py
```

### Configure Monero
1. Install Monero GUI from getmonero.org
2. Start wallet and sync
3. Dashboard auto-connects to localhost:18081
4. Customize in config.json if needed
