# l

**The minimalist letter utility**

A simple, elegant collection of words containing the letter 'l'. Because sometimes less is more.

## What is this?

`l` is a minimalist project that celebrates the letter 'l' - one of the most common consonants in the English language. This repository contains:

- A curated list of interesting words containing 'l'
- A simple command-line tool to explore these words
- Minimalist design philosophy: do one thing well

## Usage

```bash

# Get a random word
python3 l.py random

# Count words
python3 l.py count
```

In a world of complexity, `l` embraces simplicity. One letter. One purpose. Pure minimalism.
